# Power Grid Monitor

Power grid monitoring application with sensor simulation, data ingestion, anomaly detection, and predictive maintenance capabilities.

## Project Structure

```
├── sensor_simulator/
│   └── simulator.py
├── ingestion_service/
│   ├── main.py
│   └── mqtt_handler.py
├── anomaly_detection/
│   └── detector.py
├── predictive_maintenance/
│   └── predictor.py
├── common/
│   ├── models.py
│   ├── database.py
│   └── telemetry.py
├── docker-compose.yml
├── requirements.txt
└── README.md
```

## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```
